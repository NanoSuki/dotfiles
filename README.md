## Noctalia
The configuration file generated by the Noctalia GUI is located at `~/.local/state/noctalia/settings.toml`. Since most settings can be configured through the GUI, this repository does not manage Noctalia's configuration.

You can check the merged configuration with the following command:
```sh
noctalia config validate
```

To clean up the configuration and cache after uninstalling Noctalia:
```sh
rm -rf ~/.local/state/noctalia
rm -rf ~/.cache/noctalia
```
In addition, manually remove the generated application theme templates.
